# rifle-licence-test

Šiame repo pateikiamas savigynos testo klausimynas GitHub Pages aplinkai.

## Naudojimas

- Atidarykite `index.html` per GitHub Pages arba paleiskite paprastą serverį lokaliai:

```bash
cd /home/runner/work/rifle-licence-test/rifle-licence-test
python -m http.server 8000
```

- Naršyklėje atverkite `http://localhost:8000`.
- Klausimai pateikiami atsitiktine tvarka. Pasirinkus atsakymą ir paspaudus „Patikrinti“, parodoma, ar atsakymas teisingas ir koks teisingas variantas.
